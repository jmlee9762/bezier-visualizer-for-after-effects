# Bézier Visualizer for Adobe After Effects

### 🎬 Adobe After Effects ScriptUI Tool / Adobe After Effects 스크립트 툴

Visualize anchor points, Bézier handles, connector lines, and path morph animations directly inside After Effects Shape Layers.

After Effects Shape Layer의 앵커 포인트, 베지어 핸들, 연결선, 그리고 Path Morph 애니메이션을 시각화하는 스크립트입니다.

![Overview](overview.png)

## ✨ Features

🔶 Anchor Point Visualization / 앵커 포인트 시각화

🌀 Bézier Handle Visualization / 베지어 핸들 시각화

➖ Handle Connector Lines / 핸들 연결선 표시

🎨 Customizable Colors, Sizes, Shapes, and Line Widths / 색상, 크기, 형태 및 선 두께 조절

👁 Hide Zero Handles / 길이가 0인 핸들 숨김

🔄 Auto Morph Between Multiple Shape Layers / 여러 Shape Layer 간 자동 Morph 생성

🗂 Overlay Layer Generation / 오버레이 레이어 자동 생성

⚡ Overlay Update Mode / 생성된 오버레이 업데이트

## 📦 Installation

Copy the script file into:

### macOS

```txt
/Applications/Adobe After Effects/Scripts/
```

### Windows

```txt
C:\Program Files\Adobe\Adobe After Effects\Support Files\Scripts\
```

Restart After Effects and run:

```txt
File → Scripts → Bézier Visualizer
```
